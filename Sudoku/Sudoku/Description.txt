Содержание программы:

- Алгоритм автоматической генерации Судоку;
- Поле для решения вручную;
- Автоматический решатель Судоку

Правила:

В одном секторе из 9 клеток не должно быть повторяющихся цифр, каждая цифра используется ровно 1 раз.

Поле 9х9 - в каждой строке вертикальной и горизонтальной не должно быть повторяющихся цифр.

Поле задается с помощью рандомизации

Каждый из 9 секторов представляет собой массив 3х3. Класс Сектор

Метод отрисовки цифр

После этого сразу же проверяется правильность расположения по сектору, строке и столбцу


Исходный сгенерированный массив
Метод перемешивания:
- сгенерировать случайную строку, на ее основе по правилам сдвига сформировать базовую сетку
- транспонировать таблицу целиком;
- поменять строки в пределах одного сектора;
- поменять столбцы в пределах одного сектора;
- поменять районы местами по горизонтали (2 любых) и по вертикали (2 любых);
